## 基础二、null、undefined的区别？

- null 是表示无，Number(null)=0
- undefined 表示未定义，Number(undefined)=NaN

`使用场景`

* null
  
  - null，作为对象原型链的终点。
  - null，作为函数参数，表示函数的参数不是对象。


* undefined
  
  - undefined，被声明了，但是没有赋值。
  - undefined，函数没有返回值。
  - undefined，对象没有赋值属性。
  - undefined，调用函数时，应用提供的参数没有提供，该参数就是undefined。
