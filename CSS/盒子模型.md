# 盒子模型
从字面上理解就是用来包裹html标签的容器，标准盒子/怪异盒子模型都是由
content+padding+border+margin，组成
## 常见的盒子模型
* W3C盒子模型
  
  * 可通过box-sizing：content-box来设置
  * content+padding+border+margin 
* IE盒子模型
  
  * 可通过box-sizing: border-box来设置
  * content(paddind+border)+margin
